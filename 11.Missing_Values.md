# Handling Missing Values

Missing values occur when information is unavailable or not recorded.

## Common Reasons

- User skipped a field
- Data import errors
- Sensor failure
- Manual entry mistakes

## Techniques

### Remove Rows

Useful when only a few records are missing.

### Replace with Mean

Suitable for numerical columns.

Example:

Original:

20, 25, NULL, 30

Mean = 25

Updated:

20, 25, 25, 30

### Replace with Median

Preferred when data contains outliers.

### Replace with Mode

Suitable for categorical data.
