# Structured vs Unstructured Data

## Structured Data

Stored in rows and columns.

Examples

- Excel
- SQL Database

Advantages

- Easy to search
- Easy to analyze
- Fast querying

---

## Semi-Structured Data

Contains tags or metadata.

Examples

- JSON
- XML

---

## Unstructured Data

No predefined format.

Examples

- Images
- Videos
- Emails
- Audio
- Social Media Posts

---

## Comparison

| Structured | Unstructured |
|------------|--------------|
| Tables | Images |
| SQL | Videos |
| Organized | Free-form |
