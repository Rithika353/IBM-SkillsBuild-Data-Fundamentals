# Data Lifecycle

The Data Lifecycle describes how data moves through different stages.

## 1. Data Collection

Gather data from different sources.

Examples:
- Surveys
- Sensors
- Websites
- Databases

---

## 2. Data Storage

Store data securely in databases or cloud storage.

---

## 3. Data Cleaning

Remove duplicates, correct errors, and handle missing values.

---

## 4. Data Analysis

Analyze data using Excel, SQL, Python, or Power BI.

---

## 5. Data Visualization

Present insights using charts, dashboards, and reports.

---

## 6. Decision Making

Use insights to support business decisions.
