# Data Collection Methods

## Surveys

Collect responses using questionnaires.

Advantages:
- Cost-effective
- Easy to distribute
- Suitable for large audiences

---

## Interviews

Direct interaction with participants.

Advantages:
- Detailed information
- Clarification of responses

---

## Observation

Collecting data by watching behaviors or events.

Examples:
- Customer movement in stores
- Website user behavior

---

## Sensors and IoT

Devices automatically collect real-time data.

Examples:
- Smartwatches
- GPS trackers
- Temperature sensors

---

## APIs

Application Programming Interfaces allow applications to exchange data.

Examples:
- Weather API
- Google Maps API
- Stock Market API
