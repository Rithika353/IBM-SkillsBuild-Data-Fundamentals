# Data vs Information

## Data

Raw facts without context.

Example:

25
42
89

These numbers alone have no meaning.

---

## Information

Information is processed data that provides meaning.

Example:

Student Marks

Rahul - 89

Anita - 95

Riya - 78

Now the data becomes useful.

---

## Difference

| Data | Information |
|------|-------------|
| Raw facts | Processed data |
| Unorganized | Organized |
| No context | Meaningful |
| Input | Output |

---

## Example

Data:

1200

Information:

Monthly Sales = ₹1200
