# Types of Data

## Qualitative Data

Descriptive data.

Examples

- Gender
- City
- Product Category
- Customer Feedback

---

## Quantitative Data

Numerical data.

Examples

- Sales
- Age
- Revenue
- Profit

---

## Discrete Data

Countable values.

Examples

- Number of students
- Number of orders

---

## Continuous Data

Measured values.

Examples

- Height
- Weight
- Temperature
